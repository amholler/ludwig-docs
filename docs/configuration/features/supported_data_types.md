| Type       | Supported Input Feature | Supported Output Feature |
| ---------- | ----------------------- | ------------------------ |
| binary     | ✅                       | ✅                        |
| number     | ✅                       | ✅                        |
| category   | ✅                       | ✅                        |
| set        | ✅                       | ✅                        |
| bag        | ✅                       | ✅                        |
| sequence   | ✅                       | ✅                        |
| text       | ✅                       | ✅                        |
| image      | ✅                       |                          |
| timeseries | ✅                       |                          |
| audio      | ✅                       |                          |
| date       | ✅                       |                          |
| h3         | ✅                       |                          |
| vector     | ✅                       |                          |
